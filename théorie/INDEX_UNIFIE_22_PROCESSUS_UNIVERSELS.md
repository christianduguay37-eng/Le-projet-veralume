## 1. STRIC — Cycle de transformation universel

**Définition** : Toute transformation suit cinq phases : Substrat → Traitement → Résultat → Interprétation → Création. Le Substrat est la matière brute, le Traitement l'opération, le Résultat le produit, l'Interprétation le sens donné au produit, la Création la nouveauté qui émerge (Variable 5) et qui devient le Substrat du cycle suivant.

**Variables** : V1 (Substrat), V2 (Traitement), V3 (Résultat), V4 (Interprétation), V5 (Création Nouvelle)

**Connexions** : [2] Double STRIC structure deux instances de STRIC. [3] Volonté est le moteur qui oriente les transitions entre phases. [18] α_wake mesure la capacité à produire V5.

**Couche** : Cycle
